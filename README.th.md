# OkLine — เริ่มต้นใช้งาน (ภาษาไทย)

**OkLine** คือไลบรารี/SDK ภาษา Python สำหรับใช้งาน LINE ผ่าน API ของ LINE Chrome
extension — ส่งและรับข้อความ ล็อกอินด้วย **QR** หรืออีเมล และสั่งงานบัญชีของคุณเองจาก
Python ได้

> คู่มือฉบับเต็มเป็นภาษาอังกฤษ: [README.md](README.md) และโฟลเดอร์ [`docs/`](docs/index.md)
>
> Read this in English → [README.md](README.md)

> **บัญชีของคุณเอง** เท่านั้น และปฏิบัติตามข้อกำหนดการใช้งานของ LINE

## 1. ติดตั้ง

```bash
pip install okline
```

อยากให้ QR โชว์ในเทอร์มินัลแบบสวย ๆ ติดตั้งส่วนเสริม (ไม่บังคับ):

```bash
pip install "okline[qr]"
```

### สิ่งที่ต้องมีก่อน

| ต้องมี | ทำไม |
|--------|------|
| **Python 3.9 ขึ้นไป** | ใช้รันไลบรารี |
| **Node.js 18 ขึ้นไป** (อยู่ใน PATH) | ใช้คำนวณลายเซ็น `X-Hmac` ที่ LINE บังคับ (ผ่าน `ltsm.wasm`) |

ตรวจสอบว่ามี Node.js:

```bash
node --version      # ควรเป็น v18 หรือใหม่กว่า
```

## 2. เข้าสู่ระบบครั้งแรก (ทำครั้งเดียว)

มีสองทางเลือก — เลือกทางใดทางหนึ่ง:

### ก. แบบเว็บ (แนะนำสำหรับผู้เริ่มต้น / ไม่ถนัดเทอร์มินัล)

```bash
okline web                    # LinePassport
```

เบราว์เซอร์จะเปิดขึ้นที่ `http://127.0.0.1:8765` โดยอัตโนมัติ จากนั้นทำตามตัวช่วย
4 ขั้นตอน: **เริ่ม → สแกน QR → ยืนยัน PIN → เสร็จ** เปิดแอป LINE บนมือถือ
(**LINE › เพิ่มเพื่อน › QR code**) แล้วสแกน QR ที่แสดงบนหน้าจอ อ่านเพิ่มเติมที่
[docs/web.md](docs/web.md)

### ข. แบบเทอร์มินัล

```bash
okline login
```

สแกน QR ด้วยแอป LINE บนมือถือ แล้วยืนยัน PIN ที่ปรากฏ ระบบจะบันทึกเซสชัน
(รวมกุญแจ E2EE) ไว้ที่ไฟล์ `./tokens.json` — คำสั่งอื่น ๆ จะนำกลับมาใช้ให้อัตโนมัติ
โดยไม่ต้องสแกนใหม่ **เก็บไฟล์นี้เหมือนรหัสผ่าน**

## 3. ส่งข้อความแรก

จากเทอร์มินัล (หลังจากล็อกอินแล้ว):

```bash
okline send <mid> "สวัสดีจาก OkLine"
```

หรือใช้เมนูโต้ตอบ — พิมพ์ `okline` เฉย ๆ แล้วเลือกเมนูตามหมายเลข (มีทั้งหมด 8 หมวด
และรองรับการแสดงผลภาษาไทย)

จาก Python:

```python
from okline import OkLine

api = OkLine.from_tokens_file("tokens.json")   # โหลดโทเคน + กุญแจ E2EE
print(api.get_profile()["displayName"])
api.send_text("u0123456789abcdef0123456789abcdef", "สวัสดีจาก Python")
```

> `<mid>` คือรหัสผู้รับ ขึ้นต้นด้วย `u` (ผู้ใช้), `c` (กลุ่ม/แชท) หรือ `r` (ห้อง)
> ใช้ชื่อผู้ติดต่อที่ไม่ซ้ำแทนก็ได้ เช่น `okline send สมชาย "หวัดดี"`

## 4. ตั้งค่าเทอร์มินัลบน Windows

ถ้าตัวอักษรภาษาไทยหรือ QR เพี้ยนใน Command Prompt / PowerShell ให้เปลี่ยนโค้ดเพจ
เป็น UTF-8 ก่อน:

```bat
chcp 65001
```

หรือใช้ **Windows Terminal** / **PowerShell 7** ซึ่งรองรับ UTF-8 อยู่แล้ว
(ตัว CLI ของ OkLine บังคับ stdout เป็น UTF-8 ให้อยู่แล้ว หากยังเพี้ยนในสคริปต์ของ
คุณเอง ดู [docs/troubleshooting.md](docs/troubleshooting.md))
